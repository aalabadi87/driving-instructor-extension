# مدرب القيادة - Chrome Extension

## الوصف

إضافة Chrome متقدمة موجهة لمدربي ومدربات قيادة السيارات لإدارة وتتبع جدول الحصص التدريبية مع عداد تنازلي تفاعلي.

## الميزات الرئيسية

✅ **إدارة بيانات الطلاب**: حفظ بيانات الطالب (ID, الاسم, البيانات)

✅ **عداد تنازلي متقدم**: 45 دقيقة مع تنبيهات لونية

✅ **تتبع التمارين**: آخر تمرين والتمرين الحالي

✅ **مسجل الحضور**: تسجيل الحضور والغياب

## الميزات الإضافية

🌟 عرض التاريخ والساعة بالعربية

🌟 عداد مءوقت تفاعلي الوان

🌟 زر البدء/الإيقاف، الأعادة

🌟 إدارة متعددة للحصص
